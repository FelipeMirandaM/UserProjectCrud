# UserProjectCrud

Para probar este proyecto es necesario contar con JDK instalado previamente en su equipo, luego se debe clonar este proyecto y compilarlo.

Una vez compilado el proyecto habrá creado una base de datos en H2 con data previamente cargada, para ver el swagger del servicio se puede consultar el url http://localhost:8080/swagger-ui/index.html que contiene toda la información al respecto.
